# Stable Diffusion Guide

## Prompt guides

### Stable-Diffusion

For the Stable-Diffusion model, you can use natural language for the prompt to generate images
<br><br>

### Waifu-Diffusion

For the Waifu-Diffusion model, you should use Booru tags to get optimized results

### Bad examples

```
a girl with green hair wearing a deep blue hat with yellow strips and drinking a cup of tea using one english tea cup, look similar to koishi, masterpiece, clear pupils, smiling
```

this prompt is not suitable for Waifu-diffusion model because not using Booru tags.

### Good examples

```
yakumo ran, arknights, 1girl, :d, animal ears, blonde hair, breasts, cowboy shot, extra ears, fox ears, fox shadow puppet, fox tail, head tilt, large breasts, looking at viewer, multiple tails, no headwear, short hair, simple background, smile, solo, tabard, tail, white background, yellow eyes
```

this prompt is good for Waifu-Diffusion because it uses Booru tags.

### Some others recommendations

low sampling steps (10~20) can make the image look smoother, and give more freedom for further editing

Too High sampling steps(>70) may cause the image look horrible, or just too 'fake'

You may want to keep the seed when the model created a image that you like
